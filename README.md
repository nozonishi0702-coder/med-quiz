# 総合診療 知識アップデート・クイズ (MedQuiz)

総合診療医のための、問題形式の知識アップデート・復習ツール。NEJM / Lancet / Annals of Internal Medicine / JAMA / AAFP / OpenEvidence 等の新着エビデンスとレビューをもとに問題を追加しています。

- `index.html` … アプリ本体（ブラウザで開くだけ／オフライン動作）
- `questions.js` … 問題バンク（日々追加）

**免責:** 本アプリは医学教育・学習目的であり、診療上の助言ではありません。診断・治療の最終判断は各出典の一次資料と個々の患者の状況に基づいてください。
